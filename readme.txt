== Description ==

Smart Coupon, plugin to connect your wordpress store with Smart Coupon System. Just 2 step installation.
One discount coupon code can be used in multiple online stores for many times.

== Installation ==

The quickest method for installing the smart coupon is:

1. Visit plugin
1. Add new plugin
1. search smart coupon
1. Click "Install Now"
1. Finally click "Activate Plugin"

If you would prefer to do things manually then follow these instructions:

1. Upload the `smart_coupon` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the woocommerce->smart_coupon, Enter your api key and save it
